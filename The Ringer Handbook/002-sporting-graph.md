# Sporting Graph

## Executive Summary

The Sporting Graph is Ringer's core intellectual property. It models trust built through shared sporting experiences.

## Purpose

Reduce uncertainty before someone joins a game.

## Core Objects

- Player
- Connection
- Game
- Community
- Venue
- Availability
- Reputation

## Principles

- Every game strengthens the graph.
- Recommendations must be explainable.
- Trust is earned, never purchased.
- Data must improve the next game.

## Recommendation Inputs

Location, sport, skill, availability, connection strength, mutual connections, venue history, organiser reliability.

## Success

The graph succeeds when more games happen, organisers spend less time coordinating, and players confidently join new communities.
